## Project setup

Clone the repo and run an static server in the root folder

> Note: Install [static-server](https://www.npmjs.com/package/static-server) globally to deploy a simple server.

```bash
npm i -g static-server
```

Open the project's root folder and then run the static server

```bash
cd practicojs && static-server
```
